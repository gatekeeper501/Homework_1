# Helpful Resources

## Microsoft Learn
This site offers free, interactive modules that help beginners master Git branches and collaborative workflows.
[Explore Microsoft Learn](https://learn.microsoft.com/en-us/training/github/)

## GitHub Docs
The official documentation provides comprehensive guides for everything from basic commits to advanced repository management.
[Visit GitHub Docs](https://docs.github.com)

## Stack Overflow
A massive community-driven Q&A site that is essential for troubleshooting coding errors and Git issues.
[Visit Stack Overflow](https://stackoverflow.com)

## Markdown Guide
This resource provides a quick and easy reference for formatting files using Markdown syntax.
[Visit Markdown Guide](https://www.markdownguide.org)

## UA Little Rock Blackboard
This is our primary portal for accessing course materials, grades, and university announcements.
[Visit Blackboard](https://blackboard.ualr.edu)
